# kth-nla-2026

Reimplementation of the natural language autoencoder (NLA) methodology from Anthropic's [*Natural Language Autoencoders Produce Unsupervised Explanations of LLM Activations*](https://transformer-circuits.pub/2026/nla/index.html), applied to a small open-source language model.

Submission for the KTH PhD recruitment task (May–June 2026, ASSERT-KTH, supervisor: Prof. Martin Monperrus).
